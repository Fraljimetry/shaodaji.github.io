# Shaoda Ji — Academic Homepage

This is a static website for GitHub Pages and the custom domain `shaodaji.cc`.

## Features

- Light and dark themes, with the visitor’s choice saved locally
- English and Simplified Chinese interface, with the visitor’s choice saved locally
- Home, Research, Teaching, Notes, CV, and Gallery pages
- Gallery sections for a Mandelbrot-set visualization and the calligraphy work 《春江花月夜》 by 张若虚
- Responsive desktop and mobile layout

## Publish

Copy everything inside this folder to the root of the GitHub repository. `index.html` and `CNAME` must remain at the repository root. Commit to `main` and push.

The `CNAME` file contains `shaodaji.cc`.


## Course materials and formula covers

Course covers use fixed square panels and are independent from the responsive material-card grid below them. Add future course PDFs as additional `.material-card` elements without changing the cover dimensions.

All mathematical cover expressions are written as TeX inside `\[ ... \]` and rendered by MathJax. Do not replace them with Unicode or HTML approximation.
